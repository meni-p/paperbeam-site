# PaperBeam Website

This project is a documentation-style website built using VitePress for a web development lab.

## Overview

PaperBeam is a fictional content publishing platform designed to simplify writing, formatting, and content delivery.

## Features

- Custom homepage with hero section and logo
- Navigation bar with dropdown menu
- Sidebar with structured pages
- Six content pages including learning subpages
- Markdown features such as tables, lists, and code snippets
- Custom styling using CSS

## Technologies Used

- VitePress
- Markdown
- CSS

## Project Structure

- `/docs` - Contains all site pages
- `/docs/learning` - Contains learning subpages
- `/docs/.vitepress` - Configuration and theme files

## Notes

This project was created for academic purposes and demonstrates the use of VitePress for building static documentation websites.
